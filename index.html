N8N Workflows for GAP

🚨 Workflow 1: Incident Alert
// Trigger: Prometheus Webhook Prometheus Alert → N8N ↓ IF severity == "critical": ↓ Create ServiceNow Ticket (P1) ↓ Get logs from Azure (last 30min) ↓ Send logs to Claude API for analysis ↓ Send WhatsApp to on-call dev ↓ Send Teams message to #incidents ↓ Update ServiceNow with AI analysis ELSE IF severity == "warning": ↓ Create ServiceNow Ticket (P3) ↓ Send Teams message only
📈 Workflow 2: Daily Health Check
// Trigger: Cron (09:00 daily) Scheduled Trigger → N8N ↓ Query Prometheus (last 24h metrics) ↓ Query ServiceNow (open incidents) ↓ Query Azure DevOps (failed pipelines) ↓ Send data to Claude for analysis ↓ Generate health report ↓ Send to Teams "#daily-health" ↓ IF critical issues found: ↓ Alert via WhatsApp
🔄 Workflow 3: Deploy Notification
// Trigger: Azure DevOps Webhook Pipeline Complete → N8N ↓ Get deployment details ↓ IF status == "success": ↓ Send success message to Teams ↓ Update deployment dashboard ELSE: ↓ Get failure logs ↓ Claude analyzes error ↓ WhatsApp + Teams with diagnosis ↓ Create incident ticket
📝 Workflow 4: Auto-Documentation
// Trigger: Git Push to main Git Webhook → N8N ↓ Get changed files ↓ IF contains ".java" files: ↓ Send code to Claude ↓ Generate/update API docs ↓ Generate/update runbooks ↓ Update Azure Wiki ↓ Notify team via Teams
